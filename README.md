# 💊 Sentiment Analysis on Drug Reviews

## 📌 Overview

This project performs **sentiment analysis on drug reviews** using machine learning techniques. It classifies user reviews as positive or negative to help understand patient experiences and drug effectiveness.

---

## 📂 Dataset

* Contains drug reviews along with ratings and conditions.
* Used for training and evaluating sentiment classification models.

---

## ⚙️ Features

* Data preprocessing and cleaning
* Text vectorization (e.g., TF-IDF)
* Machine learning model training
* Sentiment classification (positive/negative)
* Model evaluation with accuracy metrics

---

## 🧠 Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Aashish1147/Sentiment_Analysis_On_Drugs_Review.git
   ```
2. Navigate to the project folder:

   ```bash
   cd Sentiment_Analysis_On_Drugs_Review
   ```
3. Open the notebook:

   ```bash
   jupyter notebook sentiment_analysis_final.ipynb
   ```
